## Getting Started

### Prerequisites

- Flutter (Latest Version)
- Dart SDK
- Android Studio / VS Code

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/emeleonufavour/food_store_ui.git
   ```

2. Navigate to project directory

   ```bash
   cd projectname
   ```

3. Install dependencies

   ```bash
   flutter pub get
   ```

4. Run the app
   ```bash
   flutter run
   ```
